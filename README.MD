### Description
Delivery microservice is one of five microservices responsible for processing the delivery of user orders.

Microservice functionality:
 - When a payment notification is received in a Kafka topic from the payment microservice, it creates an online delivery note via integration with the Novaya Posta API.
 - The microservice tracks changes in the delivery status of the order and publishes notifications of status changes to the Kafka topic for further processing by other services.
